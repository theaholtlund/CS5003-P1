# Guess The Cat Game

This repository contains a simple JavaScript project for a "Guess The Cat" game, where the user can guess what cat has randomly been chosen based on certain traits.

### Overview

The program is a game where players guess a cat, based on the temperament and breed information provided. Interface for the user to interact by making guesses and tracking their scores. Fetching cat information from an API, using local storage for persistent score tracking and error handling using async/await.

### Running the program

To run the program, simply open the `Assignment-1.html` file in a web browser.

### Technologies and resources

The project is based on HTML, CSS and JavaScript. External resources include the Cat API for fetching cat information and images
